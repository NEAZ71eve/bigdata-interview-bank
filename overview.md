# 任务概览：牛客网各开发岗位面经汇总

## 任务目标
从牛客网搜索各个开发岗位的面经，整理并汇总成结构化报告。

## 完成内容
通过 WebSearch 检索 + WebFetch 详情抓取，采集了 **9 大开发岗位**的牛客网面经，生成一份约 2 万字的汇总报告。

### 覆盖岗位
1. 前端开发
2. Java 后端开发
3. C++ 后端开发（200+ 道题，最详细）
4. Go 后端开发
5. 算法工程师 / 大模型方向（50 道精选题）
6. 测试开发（110 道题，含公司面试流程）
7. 大数据开发（含分技术栈专题）
8. iOS 客户端开发（43 份面经，含阿里/百度/腾讯/美团等）
9. 运维开发（京东三面完整流程）

### 报告结构
- 各岗位：技术栈范围 + 高频考点分类表 + 典型面试题 + 面试流程特点
- 跨岗位高频考点对比（通用基础 + 差异化重点 + 手撕题分布）
- 面试流程通用规律（标准流程 / 各轮侧重 / 大厂轮次参考）
- 备考策略建议（通用路径 + 各岗位专项 + 软技巧）
- 数据来源汇总（15+ 牛客网原始链接）

## 关键产出文件
- `牛客网各开发岗位面经汇总报告.md` —— 完整汇总报告

## 核心发现
- 计算机基础（TCP/进程线程/算法）是所有岗位通用底座
- 岗位差异主要体现在语言八股和领域中间件
- 大厂面试流程趋同：3 轮技术 + 1 轮 HR
