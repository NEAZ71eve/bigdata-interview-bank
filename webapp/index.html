Agent 协作流水线

5 个 Agent · 串行联动 + 阶段内并行 · 主理人中转消息 · 全程零人工干预

Phase 1
采集
collector
定向爬取牛客网大数据面经，WebSearch 补充，过滤无关岗位
→ phase1_raw_interviews.md · 36KB · 40道
Phase 2
标准化
cleaner
题型分类、去口语化、设计选项、统一编号、残缺剔除
→ phase2_standardized_questions.md · 28KB · 40道
Phase 3
扩展
generator
变式出题(4类)、派生代码题、四级难度标注、知识图谱关联
→ phase3_expanded_bank.md · 73KB · 100道
Phase 4 · 并行
阅卷
saq-grader + code-grader
简答54道(六件套) + 代码22道(五件套) 并行补全标准答案
→ phase4_saq_answers.md · 125KB
→ phase4_code_solutions.md · 107KB
Phase 5
质检
harness engine
四维量化评测：重复率/准确性/覆盖率/均衡度，PASS/FAIL + 修复清单
→ harness_report.md · ALL PASS

协作机制

机制说明
单一职责每个 Agent 只做一件事，prompt 聚焦，质量天花板高
串行联动Phase 1→2→3 严格串行，前序完成才启动后续
阶段内并行Phase 4 两个阅卷 Agent 同时工作（无依赖）
主理人中转Agent 间不直接通信，所有信息经 team-lead，避免上下文污染
阶段门禁每阶段产出经主理人核验（文件大小/行数/题目数/标签），不合格不下发
闭环修复Harness FAIL → 触发补题 → 重测 → 直到 ALL PASS

项目简介

从一份真实面经报告出发，5 个 AI Agent 协作跑完五阶段流水线，全自动产出覆盖 9 大技术栈、4 级难度、3 种题型的完整面试题库。 核心价值不在于题库本身，而在于多 Agent 协作方法论与Harness 量化评测体系。
